# Code Source Directory

This directory contains reusable Python functions for the multiwavelength image visualization notebook.

## Modules

- `archive_queries.py` - Functions for querying and downloading data from NASA archives (HEASARC, IRSA, MAST)
- `image_processing.py` - Functions for reprojection, normalization, and RGB compositing
- `plotting.py` - InteractiveRGBComposite class for adjustable RGB visualization
